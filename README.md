# SUMIKAスマホ 🎵📱

このプロジェクトは、スマートフォンで遊べる音楽クイズアプリです。HTML/CSS/JavaScript を使って構築されており、GitHub Pages を通じて公開されています。

## 🔗 公開URL
[https://ユーザー名.github.io/SUMIKAスマホ](https://ユーザー名.github.io/SUMIKAスマホ)

## 📁 ファイル構成
- `index.html` : メインページ
- `style.css` : スタイル定義
- `script.js` : クイズロジック
- `audio/` : 音声ファイル
- `images/` : 画像ファイル
- `videos/` : 動画ファイル

## 📱 スマホ対応
- レスポンシブデザイン
- タップ操作に最適化

## 🚀 GitHub Pagesでの公開方法
1. このリポジトリを `main` ブランチに push
2. GitHub の Settings → Pages で公開設定
3. 数分後にURLが有効になります
